## Built with

* React
* React router dom v6
* Vite



Clone the this  project

```bash
  git clone https://github.com/karthikhrd/super-app-1
```

Go to the project directory

```bash
  cd super-app
```

Install dependencies for super app

```bash
  npm i
```

Start the super app on the  server run the following command 

```bash
  npm run dev
```
